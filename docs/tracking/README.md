# Trip Tracking Templates

Import these CSV files to Google Sheets for easy tracking during the trip.

## Files

### 📊 expense-tracker.csv
Main spending tracker with all pre-filled estimates. Update actual costs as you go.

**Columns:**
- Date | City | Category | Item | Estimated (USD) | Actual (multiple currencies) | Status | Notes

### 🏨 hotel-estimates.csv
Quick comparison of Budget vs Mid-Range hotel options for all 6 cities.

**Shows:**
- Per-night prices for Budget and Mid-Range tiers
- Total costs per city (based on nights staying)
- Money-saving tips for mixing tiers

### 📋 booking-tracker.csv
Track what's been booked vs. estimated. Color-coded by priority.

**Priority Legend:**
- 🔴 HIGH - Book ASAP
- 🟠 BOOK SOON - 4-6 weeks ahead
- 🟡 MEDIUM - 2-4 weeks ahead
- 🟢 DONE - Already booked

### 📝 daily-log-template.csv
Simple daily journal for activities, highlights, and quick spending notes.

### 🏯 japan-traditional-stays-reviews.md
Star rating system for traditional Japanese accommodation options.

**Includes:**
- Point-based reviews (1-10 scale) across 6 categories
- Ryokans, capsule hotels, temple stays, machiyas, onsen
- "Best for" recommendations by budget and traveler type

## How to Use

1. **Open Google Sheets** → File → Import → Upload
2. **Make a copy** for each traveler if needed
3. **Use mobile app** for easy updates during trip
4. **Share** with travel companion for split expenses

## Pro Tips

- Add formulas for automatic totals
- Create pivot tables by city/category
- Add currency conversion formulas
- Export to CSV when done for backup
